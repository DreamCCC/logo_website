## 关键结论（当前阶段）

- **要做**：客户门户、注册/登录、案例与物料介绍、多步骤需求收集、瞬时「xxx 欧起」展示、数据入库  
- **暂不做**：完整自动报价算法、PDF/发票、向客户自动发邮件报价、管理后台（可列为二期）  
- **报价方式**：客户提交需求后，由人工结合工厂与物流在 **次日** 通过邮件回复正式报价  
- **界面语言**：支持 **英语 + 德语** 切换；**默认德语（DE）**  
- **注册**：注册后 **直接登录**，一期 **不做** 邮箱验证
