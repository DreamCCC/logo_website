# 03 — 设计与视觉参考

## 1. 设计方向摘要

参考提供的 **KS Retail Systems** React 首页，整体定位为：

- **深色主题**：`bg-black` / `neutral-950` 背景，白色与 neutral 灰层级文字  
- **大圆角卡片**：`rounded-[28px]` ~ `rounded-[32px]`  
- **轻字重标题**：`font-light`，大号 Hero `text-5xl` ~ `text-7xl`  
- **大图 + 渐变遮罩**：`gradient-to-t from-black/80`，悬停微缩放  
- **玻璃顶栏**：`fixed` + `backdrop-blur-xl` + `border-white/10`  
- **动效**：`framer-motion` 入场、卡片 hover 上浮  
- **图标**：`lucide-react` 风格  

竞品 [leuchtreklame-schilder.de](https://leuchtreklame-schilder.de/) 补充：

- 强调 **「在线配置 + 邮件收可视化/报价」** 的信任话术（48k+ Designs、全包、保修等）  
- **案例墙** 按材质筛选  
- **步骤条** 01 填表 → 02 报价与可视化 → 生产 → 物流 → 安装  

本项目一期：**视觉偏 KS 高端深色**；**流程话术偏竞品**（但二期才做邮件可视化）。

## 5. 多步骤询价 UI

- 顶部 **步骤指示器**：1–7，当前步高亮  
- 每步单屏主表单，底部 **Zurück / Weiter**  
- 最后一步：**摘要卡片** + 起价大号显示 + 法律勾选（Datenschutz）  
- 上传区：拖拽 Logo / 现场照片，预览缩略图  

## 6. 文案与国际化

- **双语**：界面同时支持 **英语（en）** 与 **德语（de）**；**默认 EN**；顶栏提供语言切换（如 `EN | DE`，当前语言高亮）  
- **调性**：DE 文案面向德国/欧洲客户；EN 用于国际客户，语气保持一致、专业  
- **技术规格**：mm、4000K、RAL、€ 等保持通用写法  
- **避免过度承诺**（两语均需一致）：  
  - DE：不用「sofortiges verbindliches Angebot」；可用「Unverbindliche Richtpreis ab € …」「Verbindliches Angebot per E-Mail innerhalb von 24h」  
  - EN：如 “Indicative starting price from € …”“Formal quote by email within 24 hours”  
- **注册**：无验证邮件；成功提示示例 — DE: «Konto erstellt» / EN: «Account created»
